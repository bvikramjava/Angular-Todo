Stack
-----

- Angular 4
- Angular CLI
- AngularFire2 `4.0.0-rc.1`
- Firebase
- RxJS
- SASS
- Typescript

Quick Start
-----------

$ npm install
$ npm start

|Script|Description|
|---|---|
|`npm start`|Start development server @ `localhost:4200`|
|`npm run build`|build the application to `./dist`|
|`npm run lint`|Lint `.ts` files|
